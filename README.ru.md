# ChunkWorldBuilder

English version: [README.md](README.md)

Попробуйте новый опыт игры в Minecraft: мир расширяется патч за патчем по мере исследования.

`ChunkWorldBuilder` — это Paper-плагин, который собирает целевой мир из прямоугольных патчей донорского мира.

## Что умеет

- Расширяет мир, когда игроки подходят к краю.
- Поддерживает настраиваемый размер патча (например `8x8`, `16x16`, `24x24`, `32x32`).
- Копирует ландшафт из донорского мира в целевой.
- Использует очередь и лимиты генерации для более стабильной производительности.
- Поддерживает опциональный движок копирования через WorldEdit для ускорения больших патчей.
- Может создавать прогрессионный End-портал по ачивке.
- Поддерживает защиту движения/телепорта, чтобы не пускать в недогенерированные патчи.

## Зачем играть

- Исследование ощущается живым: карта растет вместе с игроком.
- Каждый запуск может ощущаться по-разному в зависимости от донора и размеров патча.
- Подходит для выживания с вызовом, исследований и кастомной прогрессии на серверах.

## Требования

- Paper `1.21.1`
- Java `21`

## Установка

1. Соберите плагин:
   - `.\gradlew.bat clean build`
2. Поместите jar из `build/libs/` в папку `plugins/` сервера.
3. Запустите сервер.
4. При необходимости настройте `plugins/ChunkWorldBuilder/config.yml`.
5. Перезапустите сервер.

## Конфиг

Основной файл:

- `src/main/resources/config.yml` (дефолтные значения в исходниках)

Ключевые параметры:

- `generation.patch-width`
- `generation.patch-length`
- `generation.copy-engine` (`AUTO | BUKKIT | WORLDEDIT`)
- `generation.donor-range-min/max-x/z`
- `performance.max-blocks-per-tick`
- `boundary.enabled`
- `teleport-safety.enabled`

## Примечания

- При изменении размера патча в существующем мире запускается миграция сетки патчей.
- При пересоздании целевого мира данные патчей для этого мира автоматически очищаются.
